Q2. Explain one-hot encoding, does the dimensionality of the dataset increase, decrease or mains constant after applying this technique? 
Ans: One-hot encoding is a technique used to convert categorical variables into a numerical format so that machine learning models can process them.
In this method, each category is represented as a binary vector where only one value is 1 and the rest are 0.
